| ------------------ | ------------------------------------------------------------------------------------- |
| `npm run bot`      | Discord + bridge server (`createBotBridge`)                                           |

## Setup

1. `cp .env.example .env` — set `DISCORD_TOKEN` and `SHARDWIRE_SECRET`.
2. Terminal 1: `npm run bot` → wait for `bot bridge ready`.
3. Terminal 2: `npm run app` → wait for `app bridge ready`.
