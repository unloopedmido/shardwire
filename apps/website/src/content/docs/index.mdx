## What makes Shardwire different

Shardwire is not another generic event tunnel. It is built around Discord’s reality:

- gateway intents shape event availability
- scoped secrets gate each app’s action surface
- strict startup catches mismatches before your worker starts processing
- diagnostics tools explain why an event or action is unavailable
